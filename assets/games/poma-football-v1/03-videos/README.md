# Football V1 Video Assets

Bu klasörde V1 için kabul edilen dört kısa sonuç videosu bulunur:

- `goal-scored.mp4`
- `shot-missed-post.mp4`
- `save-success.mp4`
- `goal-conceded.mp4`

Uygulama, video oynatılamazsa aynı olayın `02-results` klasöründeki PNG görselini otomatik fallback olarak kullanmalıdır. Videolar sessiz/konuşmasız kullanılmalı; oyun sesleri uygulama tarafından ayrıca yönetilebilir.
